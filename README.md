# Portfolio Website

A modern, responsive portfolio website built with HTML and CSS.

## Features

- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Modern Styling** - Clean and professional design with smooth animations
- **Easy to Customize** - Simple HTML and CSS structure
- **GitHub Pages Ready** - Can be published directly on GitHub Pages

## Sections

1. **Navigation** - Sticky navbar with links to all sections
2. **Hero** - Eye-catching introduction with call-to-action
3. **About** - Tell visitors about yourself and your skills
4. **Projects** - Showcase your best work
5. **Contact** - Links to get in touch with you
6. **Footer** - Copyright and footer information

## How to Customize

### Update Your Information
Edit `index.html` and replace:
- Your name in the navigation and hero section
- Skills in the About section
- Project titles, descriptions, and links in the Projects section
- Contact links (GitHub, Email, LinkedIn, Twitter, etc.)

### Change Colors
Edit `styles.css` and modify the CSS variables at the top:
```css
:root {
    --primary-color: #0d47a1;
    --secondary-color: #1565c0;
    --accent-color: #42a5f5;
    /* ... other colors ... */
}
```

## Publishing on GitHub Pages

1. Go to your repository settings
2. Scroll to "GitHub Pages" section
3. Select "main" (or your default branch) as the source
4. Your site will be live at `https://Dunnothepeak.github.io/test/`
